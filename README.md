# YACL
Yet Another Common Library


Esta libreria apunta a tener mutliple proyectos para optimizacion y buenas practicas del ecosistema base de NET 10

Eso queire decir que cada uno de los proyectos no va a utilizar paquetes que no sean parte del stack principal propuesto en Net (Esto no cuenta para los proyecto de Benchmark o testing, solamente para el proyecto CORE que luego se convertira en un paquete nugget)

Algun proyecto pueden simplemente agregar validaciones a nivel IDE con Roslyn o crear codigo auxiliar optimo usando SourceGenerator
