# YACL
Yet Another Common Library

This library aims to include multiple projects focused on optimization and best practices for the core .NET 10 ecosystem.

This means that none of the projects will utilize packages outside of the primary stack proposed by .NET. (This restriction does not apply to Benchmark or testing projects, but strictly to the CORE project that will eventually be published as a NuGet package).

Some projects may simply add IDE-level validations using Roslyn or generate optimized auxiliary code using Source Generators.
